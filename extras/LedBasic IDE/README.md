# LedBasic IDE

Браузерная IDE для написания и тестирования скриптов на языке LedBasic.  
Открывается локально в любом современном браузере — **без сервера, без установки**.

## Запуск

Откройте файл `index.html`

## Файлы

```
LedBasic IDE/
├── index.html      — главный файл
├── script.js       — логика IDE и виртуальная машина
├── reference.js    — данные справочника языка
└── style.css       
```

## Возможности

### Редактор
- Подсветка синтаксиса: команды, функции, переменные, числа
- Нумерация строк
- Подчёркивание ошибок с tooltip
- Автодополнение (Tab / Enter / Esc)
- **Многострочный режим** — несколько операторов в одной строке:  
  `110 V = A  111 P = I  112 GOSUB 600`

### Эмулятор ленты
- Три режима: **Полоса / Кольцо / Матрица** (настраиваемые размер и масштаб)
- Отображение FPS
- Настраиваемая яркость

### Отладчик
- **Консоль** — лог компиляции и выполнения
- **Байт-код** — аннотированный hex-дамп скомпилированной программы
- **Переменные** — значения A–Z в реальном времени

### Примеры (кнопка «Примеры»)
15 готовых эффектов в трёх категориях:
- Начало: Статичный цвет, Мигание, Дыхание, Радуга
- Движение: Комета, Комета FADE, Маятник, Theater Chase
- Эффекты: Огонь, Океан, Лава, Северное сияние
- Матрица 16×16: Плазма, Лава на матрице

### Справочник (кнопка «Справочник»)
Встроенная документация с вкладками:
- **Синтаксис** — структура программы, переменные
- **Команды** — все LED и управляющие команды с примерами
- **Операторы** — арифметика, сравнение, побитовые, приоритет
- **Функции** — SIN8, COS8, EXP8, NOISE, MAP, CONSTRAIN и др.
- **Паттерны** — готовые рецепты и приёмы

### Файловые операции
- Открытие `.bas` и `.bin` файлов
- Сохранение `.bas`

### Экспорт
- `const char*` — строка для вставки в Arduino код
- `PROGMEM` — строка для хранения во Flash
- `.bas` — скачать текстовый файл
- `.bin hex` — байт-код в hex-формате

## Горячие клавиши

| Комбинация | Действие |
|---|---|
| `Ctrl+Enter` или `F5` | Запустить / Остановить |
| `Ctrl+S` | Сохранить .bas |
| `Tab` | Принять автодополнение |
| `Esc` | Закрыть автодополнение |

## Настройки

Кнопка **«Настройки»** позволяет изменить:
- Размер шрифта редактора
- Размер табуляции
- Автодополнение вкл/выкл
- Начало и шаг нумерации строк
- Скорость эмулятора (setSpeed %)
- Яркость в эмуляторе

---

## Замечания

- IDE использует JavaScript и не требует подключения к интернету.
- Виртуальная машина в браузере точно воспроизводит поведение C++ VM на ESP8266.
- Эмулятор не воспроизводит аппаратные ограничения (тайминги DMA, частота ШИМ).
- Функция `NOISE` реализована идентично C++ версии — результаты совпадают.
